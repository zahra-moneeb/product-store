### Product Store App
## A modern and responsive product management web application built with React + Vite.
This project demonstrates practical skills in frontend development, including state management, API handling, and clean UI design.
## Features
- Add new products (with image)
- View products (grid & list)
- Search products
- Product details page
- Fast data fetching with React Query

## Setup
- git clone https://github.com/your-username/product-store.git
- cd product-store
- npm install
-npm run dev
